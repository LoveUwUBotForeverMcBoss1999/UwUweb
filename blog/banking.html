The Banking!

05/08/2024

We are added a new Banking system to UwU Economy System.

1. Lot of New Features

Users can create a bank account and deposit and withdraw money. From the bank every 12 hours 2% of the bank balance adding to the bank balance.

2. Users can use these commands to use the bank:

  • uwu bsetup
  • uwu bprofile
  • uwu bbalance
  • uwu bdelete
  • uwu bwithdraw
  • uwu bdeposit
  • uwu bloan
  • uwu bloansettle
  • uwu bloaninfo

There some limits of the bank and the user balance.

User can use these commands and enjoy better UwU.
