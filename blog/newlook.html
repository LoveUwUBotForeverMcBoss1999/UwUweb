New Look Pages

06/26/2024

We're excited to unveil our freshly redesigned uwubot.online website, featuring a sleek new interface that enhances user experience and showcases UwU Bot's capabilities in style!

1. A completely new UI

We change all website UI and make new one for look and feel good.

2. A loader to get a pre-loaded page for the homepage/landing page

I find the problem about, the page takes a bit of time to load the page images like logos and other images. Most of the images are on the home page, I guess in the future. So I added a loader for the home page. So the page can be loaded.

3. A good responsive navbar

The nav bar is very responsive to the display size. And the responsive navbar has a cool/nice animation on responsive time. This responsive nav bar is 100x times better than the old responsive navbar.

By the way, if you found some bugs or glitches in the website pages, please report it from the Contact form or from our discord server using !report [the bug].

Thank you for being a part of the UwU Bot community. Your feedback and support drive us to continually improve and innovate.
