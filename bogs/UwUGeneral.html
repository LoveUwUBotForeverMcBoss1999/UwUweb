UwU General Bot

By McBoss June 8, 2024

The UwU Community Updates server got a UwU General Discord bot for monitor the server. The bot have Ticketing System, Self Role Reaction system, Welcome message from the server and DM system, Leave/Bye Bye message system Media Channel monitor system and more...

Users now can get better help from staff by making a ticket. And they can watch the Bot's status. And users can report the UwU Bot's Bugs and others from a report command (Coming Soon!)
