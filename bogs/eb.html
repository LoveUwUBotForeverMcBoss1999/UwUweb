Enable Disable

By McBoss June 8, 2024

Server Owners, Managers now can use uwu enable and uwu disable commands to Enable or Disable uwu commands for channels.

Defaultly the UwU Bot have in system Disable and Owners or Managers can use uwu enable to Enable the UwU commands to each channels.
