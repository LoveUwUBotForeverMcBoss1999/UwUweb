Features

User Balance

Every user have they own balance. They can earn UWU Money from messaging in servers that only UwU Bot in. Then you can add the UwU Bot to your server and users/members can start earn UWU Money by messaging in the server. That keep the servers more active. Users can spend UWU Money from many ways.

Hunt Wild

User can find animals, items, lootboxs from hunting. User need to pay a small fee/5 UWU Money to go on hunt. After hunting user can check the user inventory. And if animals are in inventory user can use the send_to_barn command and send those animals to the barn. And user can check the user's barn. From lootboxes user can get random things like UWU Money, Rare Animals and some items can not find from hunting!

Lottery Draw

This is a game that can be played to spend money and have fun. There are more games like this. User can buy lottery cards from lottery shop and user can visit the lottery shop using lottery_shop command. User can select a lottery and buy it. After buying the lottery bot draw the lottery by spinning the 5 numbers. If there is luck, the user can win the UWU cash prize by spinning the numbers on the lottery ticket and matching the numbers.

8-Ball

the 8-Ball command gives a random word. It is a command in the For Fun category. For the Fun category, commands do not work by taking UWU Money. these commands are for free users can use them and have some fun.
